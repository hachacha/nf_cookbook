# sept 20, 2025.

was challenged to do an overnight oats competition.
i won.

## ingredients

oats
can of coconut milk
some heavy cream
raisins
dates
apples
coconut shavings
pepitas
some honey (you don't need to)
- spices
	- cloves
	- nutmeg
	- cinnamon
	- salt :)
	- or whatever else you'd like (long pepper, ajwain, allspice, etc.)
(bbbbbbonus: ants on a log wine lees)

## method

mix oats and coconut milk and heavy cream. get a nice consistency. i would say add a little bit and then see how it soaks up. and you can always add more.
if you're using quick oats of "classic" oats it should soak up the stuff pretty quickly (prob not steel cut tho).
put cast iron skillet (or any skillet) on medium heat, put in the pepitas a few minutes later add the cloves if you're using whole cloves (use the cloves sparingly they have a lot of flavor). mind the heat, adjust as necessary and stir the things around fairly often. when the pepitas turn the slightest bit different in color add the coconut shavings and toast those until they are gold. pepitas should be making popping/cracking noises. remove the cloves or other spices you put in and grind them up. you can grind some of the pepitas and coconut in the pestle if you want.

by this time the oats should have soaked up a bit of the liquid (it's okay if the consistency is not 100% to your liking, you can add more later) so treat the rest of this like a traditional pesto:

all on the cutting board
chop up the apples pretty fine - a grater would work perfectly but not necessary.
chop the raisins and the dates not so fine
^^ we'll call this the chopped fruits (CF)
add a pile of soggy oats to the cutting board and integrate some of the CF and chop them together along with the toasted pepitas and coconut. chop all that. keep chopping. add more oats. keep chopping. add the spices. keep. chopping. until you are so over it or it's to your liking.
throw it in a bowl. if you like the consistency then go to bed. if you do not, add more of whatever you think it's missing (liquids, fruits, spices, honey, etc)

the bonus listed above was just because i had made this wine and bottled it the same day. had the leftover lees. i added it at some point during the chopping/pesto process and it definitely added a unique little flavor. i believe it would have also helped to preserve this for a bit longer as well... however i was not able to test this because it tasted too good so it only lasted like 3 days.
